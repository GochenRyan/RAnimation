# SkeletalAnimCompute Workspace

## 角色

`SkeletalAnimCompute` 是 `Samples/SkeletalAnimCompute` 的专属学习工作区。

## 对应源码范围

- `S:\Github\RAnimation\Samples\SkeletalAnimCompute`
- `S:\Github\RAnimation\Assets\SkeletalAnimCompute`

## 学习目标

- 理解 `SkeletalAnimation` CPU skinning matrix 路径如何迁移到 GPU compute。
- 理解 compute pipeline、pipeline layout、descriptor set、storage buffer、barrier 和 graphics skinning pass 的关系。
- 能解释每帧动画数据如何被打包、上传、dispatch，并最终被 vertex shader 消费。
- 能从矩阵约定、父子层级、descriptor 绑定和同步屏障角度诊断骨骼动画扭曲问题。
